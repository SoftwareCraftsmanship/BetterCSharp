# Better CSharp

C# es uno de los lenguajes de programación más populares en todo, pero pocos aprovechan completamente sus característias. C# 7 está próximo a salir así que decidí compilar una lista de las librerías, prácticas y herramientas para hacer mejor uso de C#. "Mejor" es un término subjetivo, por lo que recomiendo una lecturá muy crítica, tomar los elementos con los que te consideres cómodo y utilizarlos, en lugar de tratar de utilizar todos a la vez.
